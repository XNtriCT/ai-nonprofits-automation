# Image Prompt Rules

- The visual MUST be dictated by the specific news subject — not a default template.
- Default office/desk props (laptops, coffee, pens, notebooks, paperwork, mobile phones) are forbidden unless the news is explicitly about those things.
- Every prompt must pass this test: someone familiar with the news should recognize the subject from the visual alone, without reading the caption text overlay.
- If the news is about healthcare, the image must contain healthcare-specific elements. If space, show space. If a factory, show the factory. If a country or region, show something unique to that place.
- When in doubt, pick a visual that could ONLY belong to this specific story — never a generic business/office scene.
